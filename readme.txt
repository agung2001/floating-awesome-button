== Description ==

Floating Awesome Button (FAB) is customizable action button that can help you :
- Displaying Custom Content (Modal, Shortcodes, Widgets, Links, etc)
- Social Media Links (Facebook, Twitter, etc)

It works by generating custom modal, clickable links, and widgets inside a floating action button. You can use it in case like :
- Adding quick contact links (SMS, Email, and Call)
- Adding multiple social media links (Email, Call, Facebook)
- Displaying custom content (Offers, Opt-in, Form, etc)
- Displaying custom shortcode in a modal
- Integration with widgets in modal (Table of content, Social Sharing, etc)

= Features =

* Adding **Unlimited** custom action button
* Beautiful User Interface (UI)
* Sortable action button
* **PRO** Ability to change animation (Fade, Bounce, Zoom, Flip, etc) (25 Animation)
* **PRO** Adding custom filter by (Custom Post Type & Pages)
* **PRO** Map action button with function key (F1, F2, etc)
